# 📘 WORDLY DICTIONARY SPA

## Overview

Wordly is a Single Page Application (SPA) that allows users to search for any English word and get real time definitions, pronunciation and related information using an external Dictionary API.

The app is built using HTML, CSS, and JavaScript.

---

## Features

- Search for any English word
- Display word definitions and parts of speech
- Show pronunciation audio (when available)
- Handles empty input and invalid words
- Dynamic DOM updates without page reload
- Error handling for failed API requests

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- Async / Await
- Free Dictionary API

## How It Works

- User enters a word
- JavaScript captures input
- Fetch request is sent to the Dictionary API
- Data is processed using async/await
- Results are displayed dynamically on the page

---

## Error Handling

The application handles:
- Empty search input
- Word not found
- Network or API failures

---

This project demonstrates:
- Working with external APIs
- Async JavaScript (fetch, promises, async/await)
- DOM manipulation
- Event handling
- Error handling
- Single Page Application (SPA) behavior
